# PRD-01 采购管理模块

**页数**：约 8 页

---

## 1. 模块概述

采购管理模块覆盖从采购申请、多级审批、采购订单生成到供应商管理的全流程，实现采购业务在线化、审批流程可追溯。

---

## 2. 功能清单

| 功能编号 | 功能名称 | 优先级 | 说明 |
|----------|----------|--------|------|
| P-01 | 采购申请 | P0 | 采购员发起采购申请 |
| P-02 | 采购审批 | P0 | 多级审批流 |
| P-03 | 审批进度查询 | P0 | 申请人实时查看进度 |

| P-06 | 采购统计分析 | P2 | 采购数据统计报表 |

## 3. 功能详细说明

### 3.1 采购申请（P-01）

**角色**：采购员

**前置条件**：用户已登录，具有采购申请权限

**操作流程**：
1. 进入「采购管理」→「采购申请」
2. 点击「新建申请」
3. 填写申请信息：
   - 申请标题（必填）
   - 采购品类/商品（支持多选，必填）
   - 采购数量（必填，>0）
   - 期望供应商（下拉选择，必填）
   - 期望到货日期（必填）
   - 申请原因/备注（选填）
4. 点击「提交」→ 进入审批流程

**业务规则**：
- 同一商品同一供应商同一天不可重复申请（提示合并）
- 期望到货日期不可早于当前日期

**异常处理**：
- 供应商不存在 → 提示先维护供应商档案
- 商品不存在 → 提示先维护商品主数据

---

### 3.2 采购审批（P-02）

**角色**：采购主管、财务、总经理

**审批流程**：
```
采购员提交 → 采购主管审批 → 财务审批 → 总经理审批 → 通过
                ↓ 驳回         ↓ 驳回      ↓ 驳回
                └──────────────┴──────────┴→ 退回申请人
```

**操作流程**：
1. 进入「采购管理」→「待我审批」
2. 点击某条申请进入详情
3. 查看申请内容、历史审批意见
4. 选择操作：
   - **通过**：流转至下一审批节点
   - **驳回**：填写驳回原因，退回申请人
   - **转交**：转交给其他审批人（需填写转交原因）

**业务规则**：
- 审批顺序不可跳过（如财务未审批，总经理无法审批）
- 驳回后申请人可修改后重新提交
- 转交后原审批人不再显示该待办

**审批意见**：必填，支持富文本，最长 500 字

---

### 3.3 审批进度查询（P-03）

**角色**：采购员（申请人）

**操作流程**：
1. 进入「采购管理」→「我的申请」
2. 列表展示：申请单号、申请时间、状态、当前审批节点
3. 点击进入详情，查看：
   - 申请内容
   - 审批流程图（当前节点高亮）
   - 各节点审批人、审批时间、审批意见

**状态说明**：
- 审批中：待采购主管/财务/总经理审批
- 已通过：审批完成，已生成采购订单
- 已驳回：被驳回，可修改后重新提交
- 已撤回：申请人主动撤回（仅审批中可撤回）

---

### 3.4 供应商档案（P-04）

**角色**：采购员、采购主管、系统管理员

**字段设计**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 供应商编码 | 文本 | 是 | 唯一，系统可自动生成 |
| 供应商名称 | 文本 | 是 | |
| 联系人 | 文本 | 是 | |
| 联系电话 | 文本 | 是 | |
| 地址 | 文本 | 否 | |
| 资质证明 | 附件 | 否 | 营业执照等 |
| 合作等级 | 枚举 | 否 | A/B/C |
| 备注 | 文本 | 否 | |

**功能**：增删改查、启用/禁用、按名称/编码搜索

---

### 3.5 采购订单生成（P-05）

**触发条件**：采购申请审批流程全部通过

**系统自动执行**：
1. 生成采购订单（单号规则：PO+年月日+4位序号）
2. 订单内容来源于申请单
3. 推送通知给采购员、供应商（邮件/站内信）
4. 支持导出采购订单（PDF/Excel）

**采购订单字段**：订单号、供应商、商品明细、数量、单价、总金额、期望到货日、状态

---

### 3.6 采购统计分析（P-06）

**角色**：采购主管、总经理

**统计维度**：
- 按供应商：采购金额、采购频次、订单数
- 按品类：采购金额占比、同比/环比
- 按时间：日/周/月采购趋势

**图表**：柱状图、饼图、折线图，支持导出

---

## 4. 页面原型说明

### 4.1 采购申请列表

- 顶部：筛选（状态、时间范围）、搜索（单号）、「新建申请」按钮
- 表格：单号、申请时间、商品、数量、供应商、状态、当前节点、操作（查看/撤回）

### 4.2 采购审批详情

- 左侧：申请信息、商品明细
- 右侧：审批流程时间轴、审批意见
- 底部：通过、驳回、转交按钮

### 4.3 供应商档案

- 列表：编码、名称、联系人、电话、等级、状态、操作
- 新增/编辑：表单弹窗
