# PRD-02 仓储管理模块

**页数**：约 8 页

---

## 1. 模块概述

仓储管理模块覆盖入库、出库、库存预警、盘点、库位管理及库存报表，实现库存实时可视、预警自动触发、出入库电子化。

---

## 2. 功能清单

| 功能编号 | 功能名称 | 优先级 | 说明 |
|----------|----------|--------|------|
| W-01 | 入库管理 | P0 | 采购到货入库登记 |
| W-02 | 出库管理 | P0 | 订单出库、调拨、报损 |
| W-03 | 库存预警 | P0 | 安全库存预警与采购建议 |
| W-04 | 库存盘点 | P1 | 全盘/抽盘、盘盈盘亏 |
| W-05 | 库位管理 | P1 | 库区、货架、层架管理 |
| W-06 | 库存报表 | P2 | 库存、流水、周转率 |

---

## 3. 功能详细说明

### 3.1 入库管理（W-01）

**角色**：仓储管理员

**入库类型**：
- 采购入库：关联采购订单
- 退货入库：关联销售退货单
- 调拨入库：关联调拨单
- 其他入库：手工录入

**操作流程（采购入库）**：
1. 进入「仓储管理」→「入库管理」
2. 点击「新建入库」
3. 选择入库类型「采购入库」
4. 选择关联采购订单（自动带出商品明细）
5. 录入实际到货数量、批次号、生产日期、保质期（可选）
6. 确认入库 → 系统更新库存

**业务规则**：
- 入库数量不可超过采购订单数量
- 批次号同一商品同一批次唯一
- 入库后自动触发库存预警检测

**字段**：入库单号、入库类型、关联单号、商品、计划数量、实际数量、批次、生产日期、保质期、入库时间、操作人

---

### 3.2 出库管理（W-02）

**角色**：仓储管理员

**出库类型**：
- 销售出库：关联订单
- 调拨出库：关联调拨单
- 报损出库：手工录入原因
- 其他出库：手工录入

**操作流程（销售出库）**：
1. 进入「仓储管理」→「出库管理」
2. 点击「新建出库」
3. 选择「销售出库」，选择待出库订单
4. 系统校验库存是否充足
5. 拣货完成，录入实际出库数量
6. 确认出库 → 扣减库存

**业务规则**：
- 出库前校验库存，不足时提示并禁止出库
- 支持扫码拣货（扫描商品条码/库位码）
- 出库后若库存低于安全阈值，触发预警

---

### 3.3 库存预警（W-03）

**角色**：仓储管理员、采购员

**预警规则**：
- 当 当前库存 < 安全库存 时，生成预警
- 安全库存可按商品、仓库维度配置

**预警内容**：
- 预警商品、当前库存、安全库存、缺口数量
- 建议采购量 = 缺口数量 + 采购周期内预计消耗（可选算法）

**操作流程**：
1. 系统定时/实时检测库存
2. 触发预警 → 生成预警记录
3. 推送通知（站内信/邮件/钉钉）
4. 用户进入「库存预警」列表查看
5. 可一键发起采购申请（跳转采购模块）

**预警列表**：商品、当前库存、安全库存、缺口、建议采购量、预警时间、状态（待处理/已处理）

---

### 3.4 库存盘点（W-04）

**角色**：仓储管理员、财务

**盘点类型**：
- 全盘：全部商品
- 抽盘：按品类/库位抽样

**操作流程**：
1. 创建盘点任务，选择盘点范围、盘点日期
2. 系统生成盘点单（账面数量）
3. 盘点人实地盘点，录入实盘数量
4. 系统计算盘盈盘亏
5. 盘亏超过阈值需审批
6. 审批通过后调整库存

**业务规则**：
- 盘点期间可锁定库存（禁止出库）
- 盘盈盘亏需填写原因

---

### 3.5 库位管理（W-05）

**层级**：库区 → 货架 → 层架

**字段**：
- 库区：编码、名称、仓库
- 货架：编码、名称、所属库区
- 层架：编码、名称、所属货架、层高

**功能**：层级维护、库位与商品绑定、库位库存查询

---

### 3.6 库存报表（W-06）

**报表类型**：
- 实时库存：当前各商品库存
- 出入库流水：按时间、商品、类型筛选
- 库存周转率：按商品/品类统计

---

## 4. 页面原型说明

### 4.1 入库单

- 表头：入库类型、关联单号、入库日期
- 明细表：商品、计划数量、实际数量、批次、生产日期、保质期
- 底部：保存、确认入库

### 4.2 库存预警列表

- 筛选：仓库、品类、状态
- 表格：商品、当前库存、安全库存、缺口、建议采购量、预警时间、操作（发起采购）

### 4.3 库存查询

- 搜索：商品名称/编码
- 表格：商品、库存、安全库存、库位、最近入库/出库时间
