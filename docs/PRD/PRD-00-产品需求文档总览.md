## 二、项目背景

### 2.1 业务背景

生鲜零售企业面临采购审批效率低、库存管理粗放、配送调度依赖经验、系统权限管理不细等问题。本系统旨在通过数字化手段，打通采购-仓储-配送全链路，提升运营效率与管控能力。

### 2.2 产品目标

- **采购**：采购申请在线化、审批流程可追溯、供应商信息集中管理
- **仓储**：出入库电子化、库存实时可视、预警自动触发
- **配送**：任务智能分配、状态实时追踪、异常及时处理
- **系统**：角色权限细粒度、操作可审计、基础数据统一维护

### 2.3 目标用户

| 用户角色 | 主要职责 | 使用场景 |
|----------|----------|----------|
| 采购员 | 发起采购、跟踪到货 | 采购申请、审批进度查询 |
| 采购主管/财务/总经理 | 审批采购申请 | 多级审批、驳回/转交 |
| 仓储管理员 | 出入库、盘点、库位 | 入库登记、出库拣货、库存预警处理 |
| 配送调度员 | 分配任务、监控在途 | 任务分配、异常处理 |
| 配送员 | 接单、配送、签收 | 移动端接单、签收、异常上报 |
| 系统管理员 | 权限、用户、配置 | 角色权限、用户管理、审计查询 |
