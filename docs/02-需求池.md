## 二、采购管理需求

| ID | 需求名称 | 描述 | 优先级 | 状态 |
|----|----------|------|--------|------|
| P-001 | 采购申请 | 采购员发起采购申请，选择商品、数量、供应商、期望到货日 | P0 | 待开发 |
| P-002 | 采购审批 | 多级审批流（采购主管→财务→总经理），支持通过/驳回/转交 | P0 | 待开发 |
| P-003 | 审批进度查询 | 申请人可实时查看审批节点、审批人、审批意见 | P0 | 待开发 |
| P-004 | 供应商档案 | 供应商基本信息、资质、联系人、历史报价 | P1 | 待开发 |
| P-005 | 采购订单生成 | 审批通过后自动生成采购订单，支持导出/打印 | P1 | 待开发 |
| P-006 | 采购统计分析 | 按供应商、品类、时间维度统计采购金额与频次 | P2 | 待开发 |

## 三、仓储管理需求

| ID | 需求名称 | 描述 | 优先级 | 状态 |
|----|----------|------|--------|------|
| W-001 | 入库管理 | 采购到货后登记入库，支持批次、生产日期、保质期 | P0 | 待开发 |
| W-002 | 出库管理 | 订单出库、调拨出库、报损出库，支持扫码/手工录入 | P0 | 待开发 |
| W-003 | 库存预警 | 低于安全库存自动预警，支持阈值配置与采购建议量 | P0 | 待开发 |
| W-004 | 库存盘点 | 支持全盘、抽盘，盘盈盘亏登记与审批 | P1 | 待开发 |
| W-005 | 库位管理 | 库区、货架、层架三级库位，支持库位与商品绑定 | P1 | 待开发 |
| W-006 | 库存报表 | 实时库存、出入库流水、库存周转率报表 | P2 | 待开发 |

## 四、配送管理需求

| ID | 需求名称 | 描述 | 优先级 | 状态 |
|----|----------|------|--------|------|
| D-001 | 配送任务分配 | 按路线、运力、时效智能分配配送任务给配送员 | P0 | 部分实现 |
| D-002 | 配送状态流转 | 待接单→已接单→配送中→已签收/异常，支持状态变更与备注 | P0 | 已实现 |
| D-003 | 路线管理 | 配送路线维护，支持路线与区域、配送员绑定 | P0 | 已实现 |
| D-004 | 在途可视化 | 配送员位置实时上报，地图展示在途订单与 ETA | P1 | 待开发 |
| D-005 | 异常处理 | 配送异常登记、客户投诉处理、异常原因分类 | P1 | 待开发 |
| D-006 | 配送绩效 | 配送员准时率、签收率、投诉率统计 | P2 | 待开发 |
