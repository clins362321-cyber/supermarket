## 高保真原型

原型文件位于 `prototype/` 目录，使用 HTML + CSS 实现，可直接在浏览器中打开预览。

| 页面 | 文件 | 说明 |
|------|------|------|
| 首页工作台 | [index.html](prototype/index.html) | 待办、今日概览、快捷入口 |
| 采购管理 | [procurement.html](prototype/procurement.html) | 我的申请、审批流程 |
| 仓储管理 | [warehouse.html](prototype/warehouse.html) | 库存预警、入库、库存查询 |
| 配送管理 | [delivery.html](prototype/delivery.html) | 待配送、任务看板 |
| 系统管理 | [system.html](prototype/system.html) | 用户、角色权限、审计 |

**预览方式**：在浏览器中打开 `prototype/index.html`，通过顶部导航切换各模块。

## 四大模块与 15+ 子功能

### 采购管理
- 采购申请、采购审批、审批进度查询、供应商档案、采购订单生成、采购统计分析

### 仓储管理
- 入库管理、出库管理、库存预警、库存盘点、库位管理、库存报表

### 配送管理
- 配送任务分配、配送状态流转、路线管理、在途可视化、异常处理、配送绩效

### 系统管理
- 角色权限、用户管理、操作审计、基础数据、系统配置
